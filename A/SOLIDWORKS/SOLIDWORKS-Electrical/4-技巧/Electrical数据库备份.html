协作服务器网络配置

在 SOLIDWORKS® Electrical Schematic 应用程序中，如何配置数据库名称和实例以使用 IP 地址来避免 DNS 服务器？

方法

要在 SOLIDWORKS® Electrical、SQL 服务器和 DNS 服务器之间的协作服务器之间进行通信时节省时间并避免不必要的网络通信，请执行以下步骤：

1.在 SOLIDWORKS Electrical 中，转至“工具”>“应用程序设置”>“协作服务器”。
2.将 “地址”（服务器名称）替换为 IP 地址。请参见随附的图像“S-078403_Collaborative_server.jpg”。
3.单击“数据库”选项卡。
4.对于“服务器名称”，将 SQL Server 和实例的名称替换为服务器的 IP 地址和 SQL 使用的 TCP 端口号。使用逗号作为分隔符。例如，“127.0.0.1,1466”。请参见随附的图像“S-078403-Databases.jpg”。
5.单击“确定”并重新启动应用程序。

要确定 SQL 使用的 TCP 端口，请执行以下步骤：

1.打开 Windows® “计算机管理”应用程序。
2.展开“SQL Server 配置管理器”。
3.展开“SQL Server 网络配置”。
4.单击“[SQL 实例] 的协议”。
5.在详细信息窗格中，双击“TCP/IP”。
6.在“IP 地址”选项卡上，向下滚动以查看“IPALL”组中的“TCP 端口”。
