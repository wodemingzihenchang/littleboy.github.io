PDM安装失败

Error1722

Register_DocMgr.DLL

在继续之前，请阅读 软件要求 部分。必须安装 MSXML 和 Visual C++ Redistributable for Visual Studio 2015 才能成功注册 SOLIDWORKS Document Manager DLL。

SOLIDWORKS Document Manager API 是 SwDocumentMgr.dll 中包含的 COM 对象。SOLIDWORKS Document Manager DimXpert API 包含在 dimxpert.dll 中。如果您安装了 SOLIDWORKS 软件，则 zlib.dll DLL 会自动安装在以下文件夹中：

参考：Getting Started - 2021 - SOLIDWORKS API Help

在某些情况下，要解决此安装错误，您必须安装更新的 C++ 软件包：

https://support.microsoft.com/en-gb/help/2977003/the-latest-supported-visual-c-downloads

通常，您必须下载和安装 x86 和 x64 组件。 Visual C++ 2017 运行时组件更新 Visual C++ 2015 ，而 Visual C++ 2019 运行时组件更新 Visual C++ 2017 。 此安装是传统的，而 Visual C++ 运行时组件的早期版本是并排配置，允许每个组件的多个版本同时安装。

首先从上述链接安装这些软件包 2015-2017-2019 ，然后重试安装。
