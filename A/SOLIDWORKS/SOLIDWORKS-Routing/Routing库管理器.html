SOLIDWORKS Routing库管理器

【Routing Library Manage-电缆电线库向导】打开现有库，Routing Library Manager 包含以下项目：

Routing 零部件向导

准备用于 Routing 的零件。可以使用现有的零件或生成新零件，然后使用向导准备零件。

电缆电线向导

定义电缆电线属性（如大小、颜色和说明）的 .xml 文件。所以电气类型可将电气属性指派给线路。

创建

用于生成新的库、以 Excel 格式输入库或打开现有的库。

设置

当现在以Excel创建时，会进入设置页。遵循设置提示的“列名称对应”来导入Excel表即可。

标记方案管理器

P&ID 标签是一个字符串序列，为管道、设备或其它接头提供子标签值，例如大小或面积。 您可以使用标记方案管理器来定义管道、设备或其它接头的标记方案，以指定子标签在标签中的显示顺序。此功能在 SOLIDWORKSConnected 中不可用。

标记方案仅适用于从 P&ID 数据生成的 Routing 装配体。对电气布线似乎也不起作用。
