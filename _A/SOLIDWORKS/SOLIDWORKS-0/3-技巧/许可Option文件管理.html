Option文件管理许可

Option 文件是一个用来设置许可证服务器选项的文本文件。Options 文件可以用于各种类型的许可证：浮动许可证，临时/试用许可证，或是永久许可证。

简介

如何生成或是关联这个文件的方法根据 FLEXlm 许可证服务器的版本不同而不同。如果您需要更多的关于如何生成和关联该文件的信息，或是更多的关于 FLEXlm 许可证服务器的信息，请查看 FLEXlm 用户手册的第五章，您可以在 Macrovison 网站找到。

SolidNetwork 许可管理器的 Option 文件通常存在于 C:\Program Files (x86)\SOLIDWORKS Corp\SolidNetWork License Manager 目录下。
