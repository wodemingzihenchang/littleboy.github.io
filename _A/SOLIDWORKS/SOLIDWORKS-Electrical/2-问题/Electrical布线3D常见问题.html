Electrical布线3D常见问题

有线头无电线

自动布线不成功，或者无法布线，或者布线结束后模型连接点处有接线头但是无电线

方法：这时检查软件是否设置了默认模板，如图10所示

完成电线、电缆或线束布线过程后，布线分析对话框将列出所有错误或警告。单击 […] 可帮助您纠正错误。请参阅错误和警告列表上的可预览项。Electrical布线3D常见问题

未插入设备型号

类型： 错误

名称： 未插入设备型号

示例： “+L2+L3+F1-K1-P1 (LC1254D6)”未插入

详细信息： 设备树中所有未插入或未与SOLIDWORKS 3D 零件/装配体相关联的设备型号。

解决方法： 插入或关联设备型号。

可预览： 在 SOLIDWORKS Electrical 设备树中高亮显示设备。

已在另一个装配体中插入设备型号

类型：警告。

名称：已在另一个装配体中插入设备型号

示例：“+L2+L3+F1-K1-P1 (LC1254D6)”已插入到另一装配体

详细信息：设备树中与 SOLIDWORKS 3D 零件/装配体关联的所有电气设备在当前打开的 SOLIDWORKS 树中不可用。

解决方法：插入包含需要布线之设备型号的装配体。

可预览：在 SOLIDWORKS Electrical 设备树中高亮显示设备型号。

设备型号已与一个装配体相关联

类型：错误

名称：设备型号已与一个装配体相关联

详细信息：如果带有设备端子的电气零件与 SOLIDWORKS 装配体相关联，则会出现此情况。

备注：无法定义到 SOLIDWORKS 装配体的设备连接点。

解决方法：关联零件或将装配体保存为零件。

可预览：放大并高亮显示设备树中的 SOLIDWORKS 装配体和所选设备型号。

设备型号插入了两次

类型：错误

名称：设备型号插入了两次

详细信息：如果装配体包含与电气设备相关联的零件，且向主装配体插入两次这样的装配体，则会出现此情况。

解决方法：您可以选择消除有重复设备型号的主装配体，然后重新关联，或者移除其中一个含有重复设备型号的装配体。

可预览：在 SOLIDWORKS 中放大并高亮显示两个 3D 零件，然后在设备树中选择设备型号。

缺少设备连接点

类型：错误

名称：缺少设备连接点

详细信息：电气零件的定义与 3D 零件的定义不匹配。SOLIDWORKS 零件中缺少连接点。

备注：如果只存在一个设备连接点，则不会出现此错误。

解决方法：通过设备向导向 3D 零件添加缺少的设备连接点。

可预览：高亮显示并放大 3D 零件。

设备连接点定义无效

类型：警告

名称：设备连接点定义无效

详细信息：3D 零件的一个或多个设备连接点在电气设备的设备终端中不存在。

备注：此设备连接点无法用于布线。

解决方法：通过设备向导编辑 3D 零件，以添加缺少的设备连接点。

可预览：高亮显示设备连接点并放大 3D 零件。

只有一个设备连接点

类型：信息

名称：只有一个设备连接点

详细信息：3D 零件中只定义了一个设备连接点，并且它与设备型号定义不匹配。

解决方法：通过设备向导定义 3D 零件内部的设备连接点。

可预览：高亮显示设备连接点并放大 3D 零件。

连接点“线路类型”不是电气连接点

类型：错误

名称：设备连接点线路类型不是电气设备连接点

详细信息：如果插入至 3D 零件的非电气设备连接点与设备型号中的电气设备连接点定义相匹配，则会出现此情况。

解决方法：编辑零件并更改设备连接点布线类型。

可预览：高亮显示设备连接点并放大 3D 零件。

缺少用于电缆源/目标的 3D 零件

类型：错误

名称：缺少用于电缆源/目标的 3D 零件

定义：装配体中不存在定义为源电缆或目标的 SOLIDWORKS 零件。

解决方法：使用“设置电缆源/目标”命令可解决此问题。

可预览：无。

缺少用于电缆源/目标的 3D 零件，但已布线

类型：警告

名称：缺少用于电缆源/目标的 3D 零件，但已布线

定义：如果装配体中不存在使用“设置源/目标”命令定义为电缆源或目标的 SOLIDWORKS 零件，则会出现此情况。但是，可以使用布线方框图或多线连接对电缆布线。

解决方法：使用“设置电缆源/目标”命令可解决此问题。

可预览：无。

缺少电缆的设备连接点

类型：错误

名称：缺少电缆的设备连接点

定义：源或目标 SOLIDWORKS 零件（在电缆从/到目标、布线方框图连接或原理图连接中出现）不具有 EW_CABLE 点。

备注：如果只存在一个设备连接点，则不会出现此错误。

解决方法：通过设备向导编辑 3D 零件中的源/目标，以添加缺少的电缆点。

可预览：无。

缺少电缆的设备连接点，但已布线

类型：警告

名称：缺少电缆的设备连接点，但已布线

定义：如果源或目标 SOLIDWORKS 零件电缆从/到目标、布线方框图连接或原理图连接不具有 EW_CABLE 点，则会出现此情况。

将改为使用通过布线方框图、电缆芯等发现的包含 EW_CABLE 的 3D 零件。

备注：如果只存在一个设备连接点，则不会出现此错误。

解决方法：通过设备向导编辑 3D 零件中的源/目标，以添加缺少的电缆点。

可预览：无。

未找到有效路径

类型：警告

名称：未找到有效路径

详细信息：出现的条件：

  • 电线或电缆不存在路径。

  • 已定义 SOLIDWORKS 从/到零件，但自动布线功能无法找到有效路径。

  • 避让设置阻止使用有效路径。

解决方法：通过预览查看三色草图并识别缺少的路径零件。

可预览：预览能够使用不同颜色创建三个草图。第一张草图从第一个设备连接点开始。第二张草图从第二个设备连接点开始。第三张草图跟踪避让线段。

错误的电线属性

类型：警告

名称：错误的电线属性

定义：电线直径未定义，无法从电线截面图进行计算。

解决方法：定义截面积/规格、标准电线尺寸和电线颜色。

可预览：显示电线的“属性”对话框。

错误的电缆属性

类型：警告

名称：错误的电缆属性

定义：如果电缆直径或颜色未定义，则会出现此情况。

解决方法：定义电缆直径和颜色。

可预览：显示电缆的“属性”对话框。

3D 零件连接点的 2D 原理图针脚 ID 的定义错误

类型：错误

名称：2D 原理图针脚 ID 定义错误。

定义：连接点属性“2D 原理图针脚 ID”的值错误。对于有多个连接点的 3D 零件，只能定义一个针脚。

解决方法：删除不必要的针脚。

可预览：高亮显示设备连接点并放大 3D 零件。

错误的电缆芯属性

类型： 警告

名称： 错误的电缆芯属性。

说明： 未定义电缆芯直径

定义：未定义电缆芯的直径。

解决方法：定义电缆芯直径。使用工程电缆管理器中的替换命令。

可预览：显示具有已选电缆的电缆管理器。

与另一个连接点共线的电缆点

类型：警告

名称：与另一个连接点共线的电缆点

说明：此零件包含两个连接点共线：“”和“”

定义：在同一个零件上具有两个连接点共线时，不会产生意外的结果。类似于将多个电线和电缆芯连接至同一连接点。

解决方法：移动电缆点使其与零件的任何其他连接点均不共线。

可预览：缩放 3D 零件并突出显示出现故障的电缆点。

不完整的线束

类型：错误

名称：不完整的线束

电线说明： 电线 <电线标注>：必须将此元素添加到线束，才能为其布线。

电缆芯说明：电缆芯 <电缆标注 - 电缆芯>：必须将此元素添加到线束，才能为其布线。

定义：当连接线束设备的某些电缆/电线未添加至线束时，这些电缆/电线未布线，执行“绘制线束”。

解决方法：将提及的电缆/电线添加至线束。

可预览：缩放并突出显示电线/电缆芯连接所在线束的 3D 零件。

无法对电线布线

类型：错误

名称：无法对电线布线

电线说明： 电线 <电线标注>：无法对此元素布线，因为它已连接到属于一个线束的设备。必须将其添加到线束。

电缆芯说明：电缆芯 <电缆标注 - 电缆芯>：无法对此元素布线，因为它已连接到属于一个线束的设备。必须将其添加到线束。

定义：当某些电缆/电线连接至属于线束的设备时，这些电缆/电线未布线，执行“电线/电缆布线”。

解决方法：将提及的电缆/电线添加至线束。

可预览：缩放并突出显示电线/电缆芯连接所在线束的 3D 零件。

2D 原理图销 ID 不同于端子标注

类型：警告

名称：2D 原理图针脚 ID 不同于端子标注

说明：CPoint “<CPoint3DName@ComponentReference>”，2D 原理图针脚 ID = “<2D ID>”，端子标注 = ““

定义：连接点属性“2D 原理图针脚 ID”的值不同于在制造商零件中定义的端子的关联标注。2D 原理图销 ID 的值是：“<2D Schematic pin ID>”。端子标注的值是：。

解决方法：移除 2D 原理图针脚 ID 的当前值，然后再次布线或仅将其更改为当前值。

可预览：突出显示连接点并缩放 3D 零件，但最好是：编辑连接点特征，然后选择 2D 原理图针脚 ID 单元格。

2D 原理图销 ID 重复

类型：错误

名称：“2D 原理图销 ID”错误

说明：CPoint “<CPoint3DName@ComponentReference>”, 2D 原理图销 ID =“<2D ID>”重复

定义：部件中的多个连接点具有用户预定义的相同属性“2D 原理图销 ID”（不区分大小写）。2D 原理图销 ID 的值是：“<2D Schematic pin ID>”。

解决方法：将 2D 原理图销 ID 的当前值修改为唯一的值并再次布线，或者仅将其清除（留空），以便软件自动分配正确的值。

折弯半径错误

类型：错误

名称：折弯半径错误

说明：线路装配体中发现折弯半径错误：<线路装配体的名称>

定义：有些电线/电缆布线的线路折弯半径小于最小值。

解决方法：编辑线路装配体以手动修复几何形状。如果您不想在发现折弯半径错误时对布线的几何形状建模，请从工具 > 选项… > 布线下取消选中“如果折弯半径小于最小值，则为线段创建布线部件”选项。

连接点共享 3D 位置

类型：错误

名称：连接点共享 3d 位置

说明：<设备名称>的连接点共享相同的 3D 位置：<连接点名称>

定义：两个或多个连接点共享相同的草图点，或位于 3D 中的同一几何点。

解决方法：如果未使用连接点，则分离连接点或移除其中一个连接点。
